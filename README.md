## Tech Stack

### Frontend
- **Next.js 15.5** with App Router
- **React 19** with Server Components
- **TypeScript 5**
- **Tailwind CSS 4**
- **Turbopack** for fast builds
- **ESLint** for code quality

## Getting Started

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
